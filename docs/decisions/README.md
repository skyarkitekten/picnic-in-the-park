# Architecture Decision Records

This directory contains the Architecture Decision Records (ADRs) for the Picnic Planner project.

| ADR                                     | Title                               | Status   | Summary                                                                                |
| --------------------------------------- | ----------------------------------- | -------- | -------------------------------------------------------------------------------------- |
| [001](001-adopt-adrs.md)                | Adopt Architecture Decision Records | Accepted | Use ADRs stored in-repo to document significant architectural decisions.               |
| [002](002-azure-native.md)              | Adopt Azure-Native Architecture     | Accepted | Prefer Azure PaaS services over OSS-on-Azure or cloud-agnostic alternatives.           |
| [003](003-microsoft-foundry.md)         | Adopt Microsoft Foundry             | Accepted | Use Microsoft Foundry as the unified platform for hosting AI agents and models.        |
| [004](004-microsoft-agent-framework.md) | Adopt Microsoft Agent Framework     | Accepted | Use Microsoft Agent Framework as the sole framework for pro-code agents and workflows. |
